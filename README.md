# Lab 1  Об’єктно-орієнтоване програмування (Java)

## Тема
Новорічний подарунок
## Реалізація
У проєкті створено ієрархію класів:
- `Candy`  базовий клас  
- `ChocolateCandy`, `JellyCandy`  класи-нащадки  

Клас `NewYearGift` дозволяє:
додавати цукерки до подарунка;
обчислювати загальну вагу та вартість;
сортувати цукерки за вмістом цукру;
знаходити цукерки у заданому діапазоні.

Консольний клас `Main` містить мінімальне меню для взаємодії з користувачем.

## Структура проєкту
src/
└── ua.lab1.gift
├── app
│ └── Main.java
└── model
├── candy/
│ ├── Candy.java
│ ├── ChocolateCandy.java
│ └── JellyCandy.java
└── NewYearGift.java

test/
└── ua.lab1.gift.model
└── NewYearGiftTest.java

## Юніт-тести
Для перевірки роботи логіки використано JUnit 5.  
Тести покривають:
обчислення загальної ваги та вартості
 сортування за вмістом цукру
пошук цукерок у діапазоні

## Як запустити програму
 Запустити клас `Main.java`
 Обрати дію у консольному меню:
1) Показати всі цукерки
2) Показати загальну вагу та вартість подарунка
3) Відсортувати цукерки за вмістом цукру
4) Знайти цукерки за діапазоном вмісту цукру
